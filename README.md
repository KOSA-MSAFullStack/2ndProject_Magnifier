# 돋보기 프로젝트
시니어들을 위한 취업 지원 서비스, 돋보기 입니다.

시니어들이 쉽고 편리하게 온라인으로 취업 지원할 수 있도록 지원합니다.


## 주요 기능
- 개인/기업 회원가입
- 개인/기업 로그인
- 로그아웃
- 내 정보
- 이력서 등록 & 수정
- 이력서 상세 조회
- 전체 채용 공고 목록 조회 & 기업별 등록한 채용 공고 목록 조회
- 공고 상세 조회
- 공고 등록 & 수정 & 삭제
- 공고 지원하기
- 공고 지원내역
